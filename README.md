# jasencarroll.com

Personal site for Jasen Carroll. Static, fast, no backend.

## Stack

- [Astro](https://astro.build) — static site generation
- [Tailwind CSS v4](https://tailwindcss.com) — styling
- React islands — CursorGlow, CommandPalette (CMD+K)
- IBM Plex Mono — typography

## Dev

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Pages

- `/` — home
- `/about` — background, current work, what I'm looking for
- `/projects` — bun-stack, specz, axiomlayer
- `/resume` — full work history
- `/contact` — LinkedIn
